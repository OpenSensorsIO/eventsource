# eventsource

Server Sent Events (SSE) ELM client library (using Javascript eventsource). Influenced heavilly from the Websocket library in elm-core.

## Installation

From your top-level directory - the one with `elm-package.json` in - call:

```

$ elm package install opensensors/eventsource
```

## Documentation

[See the Elm package for full usage docs](http://package.elm-lang.org/packages/opensensors/eventsource/latest).

## Building & Testing

```
make
```

...will run the whole build and test suite.
